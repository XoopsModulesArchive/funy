Farandole
Effet complètement inutile et pas tojours du meilleur effet,
mais interessant sur le plan technique.
Récupère toutes les images de la page et les fait tourner dans une folle farandole.
Il faut affecter les nouvelles balise de ce plugin dans l'onglet "balise" du module" pensez à donner les droits en écriture sur le dossier du theme et le fichier "theme.html". Une fois affecter vous pouvez remtre les droits comme ils étaient.
