MenuRuban
Auteur : Jean-JAcques DELALANDRE
Site : http://xoops.kiolo.com

C'est un menu constitué d'images qui tournent le long d'une ellipse. Un clique sur limage permet d'accéder au lien associé.


Définition des items du menu.
La structure de chaque item est constituée de plusieurs lignes don la première doit obligatoirement être "new".
Ensuite les lignes sont constutuées d'une clé du couple de signes  ":=" (deux-points egal) et d'une valeur. Toute les clés doivent être pésentent.
Vous pouvez laisser des lignes vierges pour aérer les définitions.

img : définit l'url de l'image sans le domaine
link :  Liens à atteindre quand on clique sur l'image. le domaine doit être présent, ce qui permet de créer des liens sur des sites externes.
alt : text à afficher si l'image n'existe pas (ça ne devrait jamais arriver).
title : Texte de l'infobulle quand la souris survole une image.

exemple:

new
img:=modules/funy/_plugins/menu-ruban/images/funy.png
link:=http://xoops.kiolo.com/modules/mydownloads/viewcat.php?cid=10
alt:=Télécharger le module Funy
title:=Télécharger le module Funy

Trajectoire
La trajectoire est ue éllipse inscrite dans le rectangle défini dans les options moins les marges.
Attention de ne pas metres une trop grande large, tout le monde n'a pas encore de grand écran.
